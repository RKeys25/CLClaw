# /news
When the user triggers this command, perform the following steps:

1. **Search**: Use your search tool to find the "Latest Articles" on Binance Academy.
2. **Select**: Identify the 3 most recent articles.
3. **Summarize**: For each article, write a single paragraph (2 sentences) explaining the core topic.
4. **Format**: Display the result as a list with the Title (bolded), the summary, and a direct link.
